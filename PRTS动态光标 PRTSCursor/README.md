# P.R.T.S 动态光标

将《杀戮尖塔2》的默认鼠标箭头替换为 P.R.T.S 风格的 60 FPS 动态光标。

## 功能

- 仅替换 Arrow 光标。
- 使用预烘焙的 Idle 动画帧，并通过游戏内 overlay 播放动画，避免每帧刷新系统光标。
- 不替换点击、拖拽、IBeam、Waiting 等其他光标形态。
- 不修改卡牌、遗物、存档或战斗机制。

## 安装

将发行包中的三个文件放入游戏目录下的 `mods/PRTSCursor/`：

- `PRTSCursor.json`
- `PRTSCursor.dll`
- `PRTSCursor.pck`

## 兼容性

- 适配 Slay the Spire 2 `0.107.1`。
- 模组作者：Natsuki。
