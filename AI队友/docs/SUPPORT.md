# Support and Maintenance

AI Teammate is a hobby / experimental mod shared in a usable public state.

## Support expectations

- The mod is available for people who want to play with it, test it, or modify it.
- Bug reports and feedback may still be useful, but support may be limited.
- Long-term maintenance is not guaranteed.
- Forks and derivative work are welcome within the repository license.

## Source availability

The source code is available for users who want to inspect the implementation, tune the config files, or build their own variant.

Much of the project was built through AI-assisted development and iteration. That does not make it unusable, but it does mean future contributors should review code carefully before building on it.

## Best way to approach the mod

- Treat it as an experimental teammate system, not a polished competitive AI.
- Expect occasional strange behavior.
- If you want different personalities, start with the shipped behavior config files under `config/ai-behavior/`.
