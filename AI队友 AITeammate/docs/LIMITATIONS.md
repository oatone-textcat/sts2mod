# Limitations

AI Teammate is still an experimental mod. The current release aims for stable, understandable behavior rather than full coverage of every multiplayer or event edge case.

## What to expect from the AI

- The AI is functional, but it is not deeply strategic.
- It can make poor combat lines, awkward reward picks, or strange utility decisions.
- Different characters can still feel different because the mod ships with per-character behavior configs.

## Current release limitations

- The mod is still experimental and edge cases may remain.
- Some events have event-specific support, while others still use conservative fallback behavior.
- One risky event, `CrystalSphere`, is excluded from AI teammate event pools for release safety.
- Some multiplayer-adjacent screens and unusual room flows may still be more fragile than ordinary combats and rewards.

## Support expectations

- Config editing is supported for modders and tinkerers, but malformed files may fall back to defaults.
- The release should be treated as experimental rather than complete.
