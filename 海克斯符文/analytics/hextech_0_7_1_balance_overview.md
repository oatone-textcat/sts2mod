## 读表方式

- `持有胜率` 按最终上报时玩家实际拥有该海克斯的去重局统计，适合观察替换、合成、获得时效果后的最终相关性。
- `选择率/选择后胜率` 按海克斯选择界面候选项统计，适合观察玩家偏好和选择后的结果。
- 玩家拿到任意海克斯本身就会带来幸存者偏差，所以不要把 `选择后胜率 - 整体胜率` 当作纯强度；本报告主要用于排序候选。
- 本报告的经验阈值：强势候选优先看 `选择 >= 400 且选择后胜率 >= 78%`；弱势候选优先看 `选择 >= 300 且选择后胜率 <= 60%`；冷门候选看 `出现 >= 700 且选择率 <= 10%`。

## 建议优先级

1. 先处理断层项：玩家 `移动房车`、`不详契约`、`蛇牙`、`爆炸艺术`、`双重存在`；敌方 `缩小引擎`。

4. 0.7.1 新内容里，`逆命回响`、`破碎金冠`、`不详契约`、`双重存在` 的数据已经很强，后续新增机制最好先避免继续给同类高泛用资源。
